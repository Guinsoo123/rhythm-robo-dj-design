## 6. 现代机器人学：从姿态到动力学

### 6.1 构型空间

定义：构型 `q` 是描述机器人所有自由度位置的最小变量集合。构型空间 `C` 是所有可能构型的集合。

固定基机械臂常有：

```latex
q\in\mathbb{R}^n
```

人形机器人有浮动基，构型更接近：

```latex
q=(T_{base}, q_{joint}) \in SE(3)\times \mathbb{R}^{n}
```

机器人语境：强化学习观测不一定直接用完整构型，但必须包含足够信息判断姿态、速度和接触。

### 6.2 正运动学：指数积公式

给定各关节扭量 `S_i` 和末端初始位姿 `M`，正运动学为：

```latex
T(\theta)
=e^{\hat{S}_1\theta_1}
 e^{\hat{S}_2\theta_2}
 \cdots
 e^{\hat{S}_n\theta_n}M
```

推导直觉：第一个关节产生一个刚体变换，第二个关节在第一个变换后的坐标关系中继续作用，多个关节的运动通过群乘法组合。

机器人语境：头部、手部、脚部相对于躯干的位置，都可以由关节角通过正运动学算出。

### 6.3 速度运动学与雅可比

末端空间速度与关节速度关系：

```latex
V_s = J_s(\theta)\dot{\theta}
```

其中空间雅可比：

```latex
J_s(\theta)=
\begin{bmatrix}
S_1 &
Ad_{e^{\hat{S}_1\theta_1}}S_2 &
\cdots &
Ad_{e^{\hat{S}_1\theta_1}\cdots e^{\hat{S}_{n-1}\theta_{n-1}}}S_n
\end{bmatrix}
```

机器人语境：足端速度约束、手臂摆动速度、头部甩动速度都可以通过雅可比从关节速度得到。

### 6.4 逆运动学

目标：给定期望位姿 `T_d`，寻找关节角 `theta` 使：

```latex
T(\theta)\approx T_d
```

误差可写为：

```latex
V_b = log(T(\theta)^{-1}T_d)^\vee
```

迭代更新：

```latex
\Delta\theta = J_b(\theta)^\dagger V_b
```

```latex
\theta \leftarrow \theta + \Delta\theta
```

机器人语境：强化学习不一定显式求逆运动学，但理解它有助于设计动作空间。例如输出“关节残差”比直接输出足端目标更简单，输出足端目标则需要 IK 或全身控制器。

### 6.5 拉格朗日动力学

系统动能 `K`、势能 `P`，拉格朗日量：

```latex
L(q,\dot{q})=K(q,\dot{q})-P(q)
```

Euler-Lagrange 方程：

```latex
\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_i}
-\frac{\partial L}{\partial q_i}
=\tau_i
```

多关节机器人可写成：

```latex
M(q)\ddot{q}+C(q,\dot{q})\dot{q}+g(q)=\tau+J_c(q)^T\lambda
```

其中：

| 项 | 含义 |
| --- | --- |
| `M(q)` | 惯性矩阵 |
| `C(q,qdot)qdot` | 科氏力和离心力 |
| `g(q)` | 重力项 |
| `tau` | 关节力矩 |
| `J_c^T lambda` | 接触力映射到广义力 |

机器人语境：MuJoCo 每一步都在求解这类动力学和接触问题。强化学习策略虽然不直接写动力学方程，但它的数据来自这些方程的仿真结果。

### 6.6 接触约束

若足端接触地面且不穿透，法向间隙 `phi(q)` 满足：

```latex
\phi(q)\ge 0
```

接触力法向分量满足：

```latex
\lambda_n \ge 0
```

互补条件：

```latex
\phi(q)\lambda_n=0
```

含义：

- 如果脚离地，`phi>0`，接触力必须为 0。
- 如果脚踩地，`phi=0`，接触力可以为正。

机器人语境：人形跳舞的难点很大一部分来自接触切换。脚底接触是否稳定，会直接影响是否摔倒。

### 6.7 支撑多边形与静态稳定

定义：支撑多边形是所有接触点在地面上的凸包。

静态稳定条件近似为：重心投影落在支撑多边形内。

如果重心投影为 `p_com_xy`，支撑多边形为 `P_support`，则：

```latex
p_{com,xy}\in P_{support}
```

机器人语境：站立摇头晃脑时，动作幅度不能让重心长期离开支撑区域。更动态的动作可以短暂越界，但需要动量和落脚配合。

### 6.8 ZMP 直觉

零力矩点 ZMP 是地面上一点，使地面对机器人产生的合力矩在水平分量上为零。平地近似下，ZMP 留在支撑多边形内通常意味着动态稳定性更好。

机器人语境：第一版舞蹈控制不必显式控制 ZMP，但奖励中可以惩罚机身倾角、角速度、足端滑移，间接鼓励稳定。

### 6.9 PD 控制与 Lyapunov 稳定性

单关节二阶系统：

```latex
m\ddot{q}=\tau
```

设目标 `q^*` 为常数，误差：

```latex
e=q-q^*
```

PD 控制：

```latex
\tau=-K_p e-K_d\dot{q}
```

闭环：

```latex
m\ddot{e}+K_d\dot{e}+K_p e=0
```

选择 Lyapunov 函数：

```latex
V(e,\dot{e})=\frac{1}{2}K_p e^2+\frac{1}{2}m\dot{e}^2
```

求导：

```latex
\dot{V}=K_p e\dot{e}+m\dot{e}\ddot{e}
=\dot{e}(K_p e+m\ddot{e})
```

由闭环方程：

```latex
m\ddot{e}=-K_d\dot{e}-K_p e
```

代入：

```latex
\dot{V}=-K_d\dot{e}^2\le 0
```

所以能量不增加，系统趋向稳定。

机器人语境：本项目第一版让 RL 输出关节目标，低层 PD 负责跟踪，是因为 PD 能提供基础稳定性。RL 负责“跳什么节奏”，PD 负责“关节别失控”。

## 7. 音乐相位：把节奏变成连续控制信号

### 7.1 周期相位

设 beat 周期为 `T_b`，当前时间到最近 beat 起点的时间为 `Delta t`，相位：

```latex
\phi_b = 2\pi \frac{\Delta t}{T_b}
```

其中：

```latex
\phi_b \in [0,2\pi)
```

### 7.2 sin/cos 编码

如果直接输入 `phi_b`，在 `2pi` 回到 `0` 时会发生数值跳变。用：

```latex
z_b=[\sin\phi_b,\cos\phi_b]
```

则边界连续：

```latex
[\sin 0,\cos 0]=[0,1]
```

```latex
[\sin 2\pi,\cos 2\pi]=[0,1]
```

机器人语境：策略看到连续相位后，更容易学到周期性点头、下沉和手臂摆动，而不是在拍点边界抽动。

### 7.3 相位误差奖励

若机器人某个动作相位为 `phi_m`，音乐相位为 `phi_b`，可用圆周误差：

```latex
e_\phi = atan2(\sin(\phi_m-\phi_b), \cos(\phi_m-\phi_b))
```

奖励：

```latex
r_{phase}=\exp(-k e_\phi^2)
```

机器人语境：圆周误差避免把 `359 度` 和 `1 度` 误认为差很远。

## 8. 从机器人控制到 MDP

### 8.1 MDP 定义

马尔可夫决策过程定义为：

```latex
\mathcal{M}=(\mathcal{S},\mathcal{A},P,R,\gamma)
```

其中：

- `S`：状态空间。
- `A`：动作空间。
- `P(s'|s,a)`：转移概率。
- `R(s,a,s')`：奖励函数。
- `gamma`：折扣因子。

马尔可夫性：

```latex
P(s_{t+1}|s_t,a_t,s_{t-1},a_{t-1},...)
=P(s_{t+1}|s_t,a_t)
```

机器人语境：若观测包含 `q, qdot, base orientation, base velocity, contact, music phase, previous action`，则下一步状态近似只依赖当前观测和动作。

### 8.2 本项目 MDP

观测：

```latex
s_t=concat(s_{robot,t}, c_{music,t}, a_{t-1})
```

动作：

```latex
a_t \in [-1,1]^{n_{joint}}
```

低层目标：

```latex
q_t^*=q_{default}+\alpha clip(a_t)
```

转移：

```latex
s_{t+1}\sim P(\cdot|s_t,a_t)
```

奖励：

```latex
r_t=
r_{stability}
+r_{beat}
+r_{style}
-r_{energy}
-r_{unsafe}
```

终止：

```latex
done = fall \lor timeout \lor severe\_limit\_violation
```

### 8.3 折扣回报

定义：

```latex
G_t=\sum_{k=0}^{\infty}\gamma^k r_{t+k}
```

递推：

```latex
G_t=r_t+\gamma G_{t+1}
```

证明：

```latex
G_t
=r_t+\gamma r_{t+1}+\gamma^2r_{t+2}+\cdots
=r_t+\gamma(r_{t+1}+\gamma r_{t+2}+\cdots)
=r_t+\gamma G_{t+1}
```

机器人语境：折扣回报让策略考虑长期后果。一个动作如果当前很像跳舞但三步后摔倒，总回报仍然低。
