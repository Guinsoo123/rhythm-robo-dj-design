## 3. 微积分与优化：学习算法如何更新参数

### 3.1 导数

一元函数导数定义：

```latex
f'(x)=\lim_{h\to 0}\frac{f(x+h)-f(x)}{h}
```

直觉：导数是函数在当前位置的瞬时变化率。

机器人语境：速度是位置对时间的导数：

```latex
v(t)=\frac{dp(t)}{dt}
```

加速度是速度对时间的导数：

```latex
a(t)=\frac{dv(t)}{dt}
```

### 3.2 偏导数与梯度

多元函数 `f(x_1,...,x_n)` 对第 `i` 个变量的偏导为：

```latex
\frac{\partial f}{\partial x_i}
=\lim_{h\to 0}\frac{f(x_1,...,x_i+h,...,x_n)-f(x)}{h}
```

梯度定义为：

```latex
\nabla_x f =
\begin{bmatrix}
\frac{\partial f}{\partial x_1}\\
\vdots\\
\frac{\partial f}{\partial x_n}
\end{bmatrix}
```

性质：梯度指向函数上升最快的方向。

证明：方向导数为：

```latex
D_u f = \nabla f^T u
```

当 `||u||=1` 时，由柯西不等式：

```latex
\nabla f^T u \le ||\nabla f||_2 ||u||_2 = ||\nabla f||_2
```

等号在 `u = \nabla f / ||\nabla f||` 时取得。

机器人语境：策略梯度 `nabla_theta J(theta)` 告诉我们怎样调整网络参数，能让期望回报上升最快。

### 3.3 链式法则

若：

```latex
y=f(u), \quad u=g(x)
```

则：

```latex
\frac{dy}{dx}=\frac{dy}{du}\frac{du}{dx}
```

多元形式：

```latex
\frac{\partial L}{\partial x}
=
\left(\frac{\partial y}{\partial x}\right)^T
\frac{\partial L}{\partial y}
```

机器人语境：神经网络反向传播就是链式法则；机器人运动学中，足端位置对关节角的导数也是链式法则的空间版本。

### 3.4 雅可比矩阵

定义：向量函数 `f: R^n -> R^m` 的雅可比为：

```latex
J_f(x)=
\begin{bmatrix}
\frac{\partial f_1}{\partial x_1} & \cdots & \frac{\partial f_1}{\partial x_n}\\
\vdots & \ddots & \vdots\\
\frac{\partial f_m}{\partial x_1} & \cdots & \frac{\partial f_m}{\partial x_n}
\end{bmatrix}
```

一阶近似：

```latex
f(x+\Delta x) \approx f(x)+J_f(x)\Delta x
```

证明来自多元泰勒展开的一阶项。

机器人语境：足端位置 `p=f(q)`，则：

```latex
\dot{p}=J(q)\dot{q}
```

这说明关节速度如何合成足端速度。

### 3.5 Hessian 与二阶近似

Hessian 定义：

```latex
H_f(x) =
\begin{bmatrix}
\frac{\partial^2 f}{\partial x_1 \partial x_1} & \cdots & \frac{\partial^2 f}{\partial x_1 \partial x_n}\\
\vdots & \ddots & \vdots\\
\frac{\partial^2 f}{\partial x_n \partial x_1} & \cdots & \frac{\partial^2 f}{\partial x_n \partial x_n}
\end{bmatrix}
```

二阶泰勒展开：

```latex
f(x+\Delta x)
\approx f(x)+\nabla f(x)^T\Delta x
+\frac{1}{2}\Delta x^T H_f(x)\Delta x
```

机器人语境：Hessian 描述曲率。优化中曲率太大意味着学习率过大容易发散；机器人动力学和轨迹优化中，二阶信息可用于更快求解。

### 3.6 梯度下降与梯度上升

最小化损失 `L(theta)`：

```latex
\theta_{k+1}=\theta_k-\eta \nabla_\theta L(\theta_k)
```

最大化目标 `J(theta)`：

```latex
\theta_{k+1}=\theta_k+\eta \nabla_\theta J(\theta_k)
```

推导直觉：一阶近似：

```latex
L(\theta+\Delta\theta)
\approx L(\theta)+\nabla L(\theta)^T\Delta\theta
```

若取：

```latex
\Delta\theta=-\eta\nabla L(\theta)
```

则：

```latex
L(\theta+\Delta\theta)
\approx L(\theta)-\eta ||\nabla L(\theta)||^2
```

只要步长足够小，损失会下降。

### 3.7 约束优化与拉格朗日乘子

问题：

```latex
\min_x f(x) \quad s.t. \quad g(x)=0
```

拉格朗日函数：

```latex
\mathcal{L}(x,\lambda)=f(x)+\lambda g(x)
```

一阶必要条件：

```latex
\nabla_x f(x)+\lambda \nabla_x g(x)=0
```

直觉：在约束曲面上最优时，目标函数下降方向不能再沿着可行方向前进，因此 `nabla f` 必须与约束法向 `nabla g` 共线。

机器人语境：接触约束、关节限位、足端不滑动、安全边界都可看作约束。强化学习里常把部分约束写成奖励惩罚，但真机安全约束更应该放进 SafetyFilter。

## 4. 概率统计：随机策略与采样学习的语言

### 4.1 随机变量与概率分布

离散随机变量 `X` 的分布由概率质量函数给出：

```latex
P(X=x_i)=p_i,\quad \sum_i p_i=1
```

连续随机变量由概率密度函数给出：

```latex
P(a \le X \le b)=\int_a^b p(x)dx
```

机器人语境：训练时动作、初始姿态、摩擦系数、音乐片段都可以是随机变量。随机性让策略不只适应一个固定场景。

### 4.2 期望、方差与样本均值

期望：

```latex
E[X]=\sum_x p(x)x
```

连续情形：

```latex
E[X]=\int xp(x)dx
```

方差：

```latex
Var(X)=E[(X-E[X])^2]
```

等价形式：

```latex
Var(X)=E[X^2]-E[X]^2
```

证明：

```latex
E[(X-\mu)^2]
=E[X^2-2\mu X+\mu^2]
=E[X^2]-2\mu E[X]+\mu^2
=E[X^2]-\mu^2
```

其中 `mu=E[X]`。

采样估计：

```latex
E[X]\approx \frac{1}{N}\sum_{i=1}^{N}X_i
```

机器人语境：强化学习最大化的是“很多次试跳后的平均长期表现”，不是单次运气好的动作。

### 4.3 条件概率与贝叶斯公式

条件概率：

```latex
P(A|B)=\frac{P(A,B)}{P(B)}
```

贝叶斯公式：

```latex
P(A|B)=\frac{P(B|A)P(A)}{P(B)}
```

机器人语境：给定当前状态和动作，下一状态的分布为：

```latex
P(s_{t+1}|s_t,a_t)
```

这是 MDP 的动力学核心。仿真中它由 MuJoCo 隐式给出；真实世界中它由物理、接触、传感噪声和控制延迟共同决定。

### 4.4 高斯分布

一维高斯分布：

```latex
p(x)=\frac{1}{\sqrt{2\pi}\sigma}
\exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)
```

对数概率：

```latex
\log p(x)
=-\frac{1}{2}\left[
\frac{(x-\mu)^2}{\sigma^2}
+2\log\sigma+\log(2\pi)
\right]
```

对角多维高斯：

```latex
\pi_\theta(a|s)=\mathcal{N}(\mu_\theta(s), diag(\sigma_\theta^2(s)))
```

机器人语境：连续控制常用高斯策略。均值是“推荐动作”，方差是“探索幅度”。部署真机时通常使用均值动作而不是采样动作。

### 4.5 熵

离散分布熵：

```latex
H(P)=-\sum_x p(x)\log p(x)
```

连续分布微分熵：

```latex
H(p)=-\int p(x)\log p(x)dx
```

高斯分布熵：

```latex
H(\mathcal{N}(\mu,\sigma^2))
=\frac{1}{2}\log(2\pi e\sigma^2)
```

机器人语境：熵越大，策略探索越强。早期训练加熵奖励可以避免机器人太早学成“站着不动最安全”。

### 4.6 KL 散度

定义：

```latex
D_{KL}(P||Q)=\sum_x P(x)\log\frac{P(x)}{Q(x)}
```

连续情形：

```latex
D_{KL}(P||Q)=\int p(x)\log\frac{p(x)}{q(x)}dx
```

性质：`D_KL(P||Q) >= 0`。

证明来自 Jensen 不等式。令 `Z=Q(X)/P(X)` 且 `X~P`：

```latex
D_{KL}(P||Q)
=E_P\left[-\log\frac{Q(X)}{P(X)}\right]
\ge -\log E_P\left[\frac{Q(X)}{P(X)}\right]
=-\log\sum_x Q(x)=0
```

机器人语境：PPO 希望新旧策略不要差太远，KL 散度可以度量策略更新幅度。策略更新太猛，机器人可能突然从稳定动作跳到危险动作。

## 5. 李群与李代数：三维刚体运动的语言

### 5.1 为什么需要李群

三维旋转不能简单当作普通向量相加。两个旋转矩阵相乘仍是旋转，但两个旋转矩阵相加通常不是旋转。机器人姿态、机身朝向、足端位姿都生活在特殊几何空间上，这就是李群语言的必要性。

### 5.2 群

定义：集合 `G` 配合运算 `*` 称为群，若满足：

- 封闭性：`a,b in G => a*b in G`。
- 结合律：`(a*b)*c=a*(b*c)`。
- 单位元：存在 `e`，使 `e*a=a*e=a`。
- 逆元：对任意 `a`，存在 `a^{-1}`，使 `a*a^{-1}=e`。

机器人语境：旋转矩阵集合 `SO(3)` 在矩阵乘法下构成群；刚体变换集合 `SE(3)` 也构成群。

### 5.3 SO(3)

定义：

```latex
SO(3)=\{R\in\mathbb{R}^{3\times3}\mid R^TR=I,\ det(R)=1\}
```

证明 `SO(3)` 对乘法封闭：若 `R_1,R_2 in SO(3)`，则：

```latex
(R_1R_2)^T(R_1R_2)
=R_2^TR_1^TR_1R_2
=R_2^TR_2
=I
```

且：

```latex
det(R_1R_2)=det(R_1)det(R_2)=1
```

所以 `R_1R_2 in SO(3)`。

机器人语境：机身姿态、头部朝向、足端朝向都属于 `SO(3)`。

### 5.4 反对称矩阵与 so(3)

对向量 `omega=[omega_1,omega_2,omega_3]^T`，定义帽算子：

```latex
\hat{\omega}=
\begin{bmatrix}
0 & -\omega_3 & \omega_2\\
\omega_3 & 0 & -\omega_1\\
-\omega_2 & \omega_1 & 0
\end{bmatrix}
```

它满足：

```latex
\hat{\omega}v=\omega \times v
```

李代数：

```latex
so(3)=\{\hat{\omega}\in\mathbb{R}^{3\times3}\mid \hat{\omega}^T=-\hat{\omega}\}
```

机器人语境：角速度 `omega` 可以通过 `hat(omega)` 表示成矩阵，从而描述旋转随时间变化。

### 5.5 旋转矩阵微分

若 `R(t) in SO(3)`，则：

```latex
R(t)^TR(t)=I
```

对时间求导：

```latex
\dot{R}^TR+R^T\dot{R}=0
```

所以：

```latex
R^T\dot{R}
```

是反对称矩阵，可写成：

```latex
R^T\dot{R}=\hat{\omega}
```

因此：

```latex
\dot{R}=R\hat{\omega}
```

机器人语境：IMU 角速度与姿态变化之间的关系就依赖这个公式。

### 5.6 指数映射 SO(3)

定义：

```latex
R=\exp(\hat{\omega}\theta)
```

Rodrigues 公式：

```latex
\exp(\hat{\omega}\theta)
=I+\sin\theta \hat{\omega}
+(1-\cos\theta)\hat{\omega}^2
```

其中 `||omega||=1`。

直觉：李代数中的小旋转向量通过指数映射变成真正的旋转矩阵。

机器人语境：策略或控制器可以在局部用三维旋转误差向量表示姿态误差，但实际更新姿态时要回到 `SO(3)`。

### 5.7 SE(3)

刚体变换定义：

```latex
T=
\begin{bmatrix}
R & p\\
0 & 1
\end{bmatrix}
\in SE(3)
```

其中 `R in SO(3)`，`p in R^3`。

变换点：

```latex
\bar{x}_{world}=T\bar{x}_{body}
```

其中齐次坐标：

```latex
\bar{x}=
\begin{bmatrix}
x\\1
\end{bmatrix}
```

逆变换：

```latex
T^{-1}=
\begin{bmatrix}
R^T & -R^Tp\\
0 & 1
\end{bmatrix}
```

证明：

```latex
\begin{bmatrix}
R & p\\0 & 1
\end{bmatrix}
\begin{bmatrix}
R^T & -R^Tp\\0 & 1
\end{bmatrix}
=
\begin{bmatrix}
I & -RR^Tp+p\\0 & 1
\end{bmatrix}
=I
```

机器人语境：人形机器人的躯干、头、手、脚之间的相对位姿都可以用 `SE(3)` 表示。

### 5.8 se(3)、扭量与指数积

空间速度或扭量：

```latex
V=
\begin{bmatrix}
\omega\\
v
\end{bmatrix}
```

对应矩阵：

```latex
\hat{V}=
\begin{bmatrix}
\hat{\omega} & v\\
0 & 0
\end{bmatrix}
```

刚体运动可写为：

```latex
T(\theta)=\exp(\hat{V}\theta)
```

机器人语境：旋转关节和移动关节都可以统一写成扭量。现代机器人学中的 Product of Exponentials 公式就建立在这里。
