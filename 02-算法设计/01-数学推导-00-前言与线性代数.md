# 数学与学习推导 · 前言、记号与线性代数

本分册属于《数学与学习定义证明推导》系列。总索引见 [01-数学与学习定义证明推导](01-数学与学习定义证明推导.md)。


本文是一篇可以独立阅读的“学习与推导章节”。它从线性代数、微积分、概率统计、李群李代数开始，逐步推到现代机器人学中的运动学、动力学、稳定控制，再推到马尔可夫决策过程、Bellman 方程、策略梯度、Actor-Critic、GAE 与 PPO。目标是让读者能从全局上理解：为什么本项目可以把“人形机器人随摇滚音乐跳舞”写成一个强化学习控制问题。

本文不追求数学教材级别的完备性，但会尽量做到三件事：

- 给出每个核心对象的定义。
- 写出关键公式从哪里来。
- 解释它在机器人强化学习中的作用。

## 0. 全局理论路线

机器人强化学习不是孤立算法，而是几条理论线索汇合后的工程系统：

```text
线性代数
  -> 状态、动作、误差、网络权重、雅可比、惯性矩阵

微积分与优化
  -> 梯度、链式法则、泰勒展开、梯度下降、约束优化

概率统计
  -> 随机策略、期望回报、采样估计、熵、KL 散度

李群与李代数
  -> 三维旋转、刚体位姿、速度扭量、指数映射

现代机器人学
  -> 正运动学、速度运动学、动力学、接触、稳定性、PD 控制

强化学习
  -> MDP、Bellman 方程、价值函数、策略梯度、PPO

项目落地
  -> 音乐条件观测 + 人形机器人控制 + MuJoCo 训练 + 安全部署
```

本项目最终训练的策略可抽象为：

```latex
a_t \sim \pi_\theta(a_t \mid s_t)
```

其中 `s_t` 包含机器人本体状态、音乐节拍相位、上一帧动作等信息；`a_t` 是关节位置残差、速度目标或力矩等动作；`theta` 是神经网络参数。训练目标是最大化期望折扣回报：

```latex
J(\theta)=E_{\tau \sim \pi_\theta}\left[\sum_{t=0}^{T}\gamma^t r_t\right]
```

下面从支撑这个目标函数的数学基础开始推导。

## 1. 记号约定

| 记号 | 含义 |
| --- | --- |
| `x, y` | 向量或随机变量，视上下文而定 |
| `A, B, M` | 矩阵 |
| `q` | 机器人广义坐标，常表示关节角和浮动基位姿 |
| `qdot` | 广义速度 |
| `tau` | 广义力或关节力矩；在 RL 章节也可表示轨迹，具体看上下文 |
| `R` | 三维旋转矩阵 |
| `T` | 刚体变换矩阵，属于 `SE(3)` |
| `s_t` | 第 `t` 步状态或观测 |
| `a_t` | 第 `t` 步动作 |
| `r_t` | 第 `t` 步奖励 |
| `pi_theta(a|s)` | 参数为 `theta` 的策略 |
| `P(s'|s,a)` | 环境转移概率 |
| `gamma` | 折扣因子，`0 <= gamma <= 1` |
| `E[...]` | 期望 |

## 2. 线性代数：状态、误差与坐标变换的语言

### 2.1 向量空间

定义：实向量空间 `R^n` 是所有长度为 `n` 的实数向量集合，并满足向量加法和标量乘法封闭。

```latex
x =
\begin{bmatrix}
x_1\\
x_2\\
\vdots\\
x_n
\end{bmatrix}
\in \mathbb{R}^n
```

机器人语境：关节角 `q`、关节速度 `qdot`、IMU 角速度、音乐特征、策略动作都可以拼接成向量。

如果人形机器人有 `n` 个可控关节，第一版动作可写成：

```latex
a_t \in [-1,1]^n
```

再缩放为关节目标：

```latex
q_t^* = q_{default} + \alpha a_t
```

这就是用向量描述全身动作。

### 2.2 线性组合、基与坐标

定义：向量 `x` 能写成基向量 `e_i` 的线性组合：

```latex
x = \sum_{i=1}^{n} x_i e_i
```

其中 `x_i` 是坐标。

直觉：坐标不是物体本身，而是“在某套尺子下的表达”。同一个足端速度，在世界坐标系和机身坐标系中的数值不同，但物理运动相同。

机器人语境：策略观测中经常把世界系重力方向投影到机身系：

```latex
g_{body} = R_{body}^{T} g_{world}
```

这样策略看到的是“我身体相对重力歪了多少”，比直接看世界坐标更稳定。

### 2.3 内积与角度

定义：两个向量的内积为：

```latex
x^T y = \sum_{i=1}^{n} x_i y_i
```

内积与夹角关系：

```latex
x^T y = ||x||_2 ||y||_2 \cos\theta
```

推导：由余弦定理：

```latex
||x-y||_2^2 = ||x||_2^2 + ||y||_2^2 - 2||x||_2||y||_2\cos\theta
```

又有：

```latex
||x-y||_2^2 = (x-y)^T(x-y)=x^Tx+y^Ty-2x^Ty
```

两式比较，得到：

```latex
x^Ty = ||x||_2||y||_2\cos\theta
```

机器人语境：若要奖励机身朝向与目标方向一致，可用内积：

```latex
r_{heading} = \hat{h}_{body}^{T}\hat{h}_{target}
```

当方向一致时奖励接近 1，垂直时接近 0，相反时接近 -1。

### 2.4 范数与误差惩罚

定义：欧氏范数：

```latex
||x||_2 = \sqrt{x^Tx}
```

平方范数：

```latex
||x||_2^2 = x^Tx
```

对 `x` 求梯度：

```latex
\nabla_x (x^Tx) = 2x
```

证明：展开为：

```latex
x^Tx = \sum_i x_i^2
```

对每个分量求偏导：

```latex
\frac{\partial}{\partial x_i}\sum_j x_j^2 = 2x_i
```

所以梯度向量是 `2x`。

机器人语境：动作惩罚常写成：

```latex
r_{action} = -w_a ||a_t||_2^2
```

它会惩罚大幅度动作。因为梯度与动作本身成正比，动作越大，训练中被拉回的力度越大。

### 2.5 矩阵与线性映射

定义：矩阵 `A in R^{m x n}` 表示从 `R^n` 到 `R^m` 的线性映射：

```latex
y = Ax
```

线性映射满足：

```latex
A(\alpha x + \beta y)=\alpha Ax+\beta Ay
```

机器人语境：

- 神经网络线性层：`z = Wx + b`。
- 雅可比速度映射：`v = J(q)qdot`。
- 坐标旋转：`p_{world}=Rp_{body}`。

### 2.6 正交矩阵与旋转

定义：若矩阵 `R` 满足：

```latex
R^TR=I
```

则 `R` 是正交矩阵。若进一步满足：

```latex
det(R)=1
```

则 `R` 是旋转矩阵，属于 `SO(3)`。

性质：正交矩阵保持长度。

证明：

```latex
||Rx||_2^2=(Rx)^T(Rx)=x^TR^TRx=x^Tx=||x||_2^2
```

机器人语境：旋转只改变坐标表达，不应该改变向量长度。例如一个重力向量从世界系变到机身系，重力大小仍然是 `9.81`。

### 2.7 投影

定义：向量 `x` 在单位向量 `u` 上的投影为：

```latex
proj_u(x) = (u^Tx)u
```

若 `u` 不是单位向量：

```latex
proj_u(x)=\frac{u^Tx}{u^Tu}u
```

证明思路：投影向量应为 `alpha u`，误差 `x-alpha u` 与 `u` 正交：

```latex
u^T(x-\alpha u)=0
```

解得：

```latex
\alpha=\frac{u^Tx}{u^Tu}
```

机器人语境：判断机身倾斜时，可把重力方向投影到机身坐标轴；判断足端是否滑动时，可把足端速度分解为法向和切向分量。

### 2.8 特征值、正定矩阵与能量

定义：若存在非零向量 `v` 和标量 `lambda`，使：

```latex
Av=\lambda v
```

则 `lambda` 是特征值，`v` 是特征向量。

定义：对称矩阵 `M` 若对任意非零 `x` 都满足：

```latex
x^TMx > 0
```

则称 `M` 正定。

机器人语境：惯性矩阵 `M(q)` 通常是对称正定的，因为动能：

```latex
K = \frac{1}{2}qdot^T M(q) qdot
```

必须对任意非零速度为正。这保证“运动需要能量”，也保证动力学方程在数值上更可解。

### 2.9 最小二乘

问题：给定 `Ax ≈ b`，寻找让误差最小的 `x`：

```latex
\min_x ||Ax-b||_2^2
```

推导：

```latex
f(x)=(Ax-b)^T(Ax-b)
```

展开：

```latex
f(x)=x^TA^TAx-2b^TAx+b^Tb
```

求梯度并令零：

```latex
\nabla_x f = 2A^TAx-2A^Tb=0
```

得到正规方程：

```latex
A^TAx=A^Tb
```

机器人语境：逆运动学、系统辨识、传感器标定和动作拟合都常出现最小二乘。若某个足端目标位置无法精确达到，就找误差最小的关节变化。
